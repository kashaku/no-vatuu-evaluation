# Vatuu Killer

这是一个 Manifest V3 Chrome 插件，用于在用户点击“开始”后自动进入 VATUU 课程评价问卷、填写题目并提交。

## 使用方式
首先git clone源代码，即下载本仓库。
1. 打开 Chrome 扩展程序页面：`chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目目录：`\no-vatuu-evaluation`
5. 登录 VATUU，为途教学信息服务平台，进入“课程评价”列表页
6. 点击插件图标，设置评分、问答题文本和提交等待时间，然后点“开始”

## 行为说明

- 插件会在所有 frame 中注入脚本，因此能处理 VATUU 桌面壳中的 iframe 页面。
- 课程评价列表页会自动打开第一条“填写问卷”。
- 问卷页会逐题选择指定分数，并填写文本题。
- 默认提交前等待 15 秒；若提交失败，可以尝试增大等待时间。
- 如果提交后同一份问卷仍出现在列表中，插件会停止并提示手动检查。

## 本地样例

项目中的两个 HTML 快照可用于核对 DOM 结构。若要让插件作用于本地 `file://` 页面，需要在 Chrome 扩展详情页开启“允许访问文件网址”。
